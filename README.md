# UCKIM - Université Chretienne de KIMPESE (Refactored)

This project contains the frontend source code for the UCKIM platform, refactored into a professional and maintainable structure.

## Structure du Projet

- **`html/`** : Contient toutes les pages HTML individuelles de l'application (Accueil, Facultés, Forum, Inscription, Portails).
- **`css/`** : 
  - `main.css` : Styles globaux, variables, typographie.
  - `header.css` : Styles pour la barre de navigation.
  - `components/` : Styles réutilisables (boutons, formulaires, cartes, tables, alertes).
  - `pages/` : Styles spécifiques à chaque page.
- **`js/`** :
  - `main.js` : Scripts globaux (navigation, animations).
  - `data.js`, `programmes.js` : Données globales de l'application.
  - `components/` : Scripts pour des composants partagés (ex: calculs de cotes).
  - `pages/` : Logique JavaScript spécifique aux pages individuelles.
- **`assets/`** : Dossier principal pour les ressources statiques.
  - `images/`, `icons/`, `videos/`, `fonts/`.
